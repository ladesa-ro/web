# Iniciar o SISGHA em sua máquina.

```bash
git clone https://github.com/sisgha/sisgha-app.git;
cd sisgha-app;
```

## Setup

Instale as dependências do projeto:

```bash
npm install
```

## Servidor de desenvolvimento

Comece o servidor de desenvolvimento em `http://localhost:3000`:

```bash
npm run dev
```
